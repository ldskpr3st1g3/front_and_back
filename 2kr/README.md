# Контрольная работа №2

## Фронтенд и бэкенд разработка

Итоговый проект по практическим занятиям **7–11**.  
В рамках работы было разработано fullstack-приложение: **магазин товаров** с авторизацией, refresh-токенами, ролями пользователей и React-интерфейсом.

## Что реализовано

### Backend

- Node.js + Express
- регистрация и вход пользователя
- JWT access-token
- refresh-token
- обновление токенов через `/api/auth/refresh`
- middleware для проверки авторизации
- role-based access control (RBAC)
- CRUD для товаров
- управление пользователями для администратора
- Swagger-документация API

### Frontend

- React + Vite
- страницы:
  - регистрация
  - вход
  - магазин товаров
  - создание/редактирование товара
  - подробная карточка товара
  - управление пользователями
- защищенные маршруты
- Axios interceptors
- автоматическое обновление access-token через refresh-token
- карточки товаров с изображениями
- предпросмотр изображения по URL

## Роли пользователей

- **user** — просмотр товаров
- **seller** — просмотр, создание и редактирование товаров
- **admin** — полный доступ, включая удаление товаров и управление пользователями

## Что изучено

В ходе выполнения работы были изучены:

- создание REST API на Express
- JWT-аутентификация
- refresh-токены и их ротация
- middleware и защита маршрутов
- React Router
- Axios interceptors
- работа с localStorage
- CRUD-операции
- ролевая модель доступа (RBAC)
- интеграция frontend и backend
- документирование API через Swagger
