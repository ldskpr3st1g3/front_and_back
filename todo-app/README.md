# PWA Notes

## Описание

Прогрессивное веб-приложение для создания заметок и напоминаний с push-уведомлениями, офлайн-режимом и возможностью установки на устройство.

## Технологии

- **Node.js + Express** — сервер
- **Socket.IO** — реалтайм-связь клиент-сервер
- **Web Push (VAPID)** — push-уведомления
- **Service Worker** — кэширование, офлайн, обработка push
- **Web App Manifest** — установка как нативное приложение
- **LocalStorage** — хранение заметок на клиенте

## Структура проекта

├── server.js # Сервер (Express, Socket.IO, Web Push)
├── index.html # Главная страница
├── app.js # Клиентская логика
├── sw.js # Service Worker
├── style.css # Стили
├── manifest.json # Манифест PWA
├── content/
│ ├── home.html # Страница заметок
│ └── about.html # Страница «О приложении»
├── icons/ # Иконки приложения
├── key.pem # SSL ключ
└── cert.pem # SSL сертификат

## Функционал

- **Создание и удаление заметок**
- **Напоминания с выбором даты и времени**
- **Push-уведомления (с кнопкой «Отложить на 5 минут»)**
- **Всплывающий popup при срабатывании напоминания**
- **Работа офлайн (кэширование через Service Worker)**
- **Установка на устройство как PWA**
- **Подписка/отписка от уведомлений**
